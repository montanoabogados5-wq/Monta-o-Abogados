# Montaño Abogados | Consorcio Jurídico e Inmobiliaria

## Requisitos
- Node.js 18 o 20

## Instalar
npm install

## Ejecutar
npm run dev

## Notas
- Configuración Tailwind lista con @tailwindcss/postcss
- Imágenes: public/hero.jpg y public/services/*.jpg
- Edita servicios y marca 'img' en src/App.jsx (SERVICES)
